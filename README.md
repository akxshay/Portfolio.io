# Akshay Kumar S — Portfolio

A terminal/IDE-styled developer portfolio built with React + Vite.

## Run it locally

You need [Node.js](https://nodejs.org) installed (v18 or newer — check with `node -v`).

```bash
# 1. Unzip this folder, then open a terminal inside it
cd my-portfolio

# 2. Install dependencies
npm install

# 3. Start the dev server
npm run dev
```

Then open the URL it prints (usually **http://localhost:5173**) in your browser.

## Build for deployment (e.g. Vercel, Netlify, GitHub Pages)

```bash
npm run build
```

This creates a `dist/` folder with static files you can deploy anywhere.

## Editing content

Open `src/Portfolio.jsx` — your name, bio, skills, education, and the `PROJECTS`
array near the top of the file are all plain JS/JSX, easy to find and edit.
